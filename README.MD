### z-py

  > Always With Me, Function!.

    1. function nowf
    2. class MYSQL 

### Install:

    pip3 install -U z-py

### Usage:

- Method 1
  ```python
  import zpy
  print(zpy.nowf('t')) #format now()!
  ```

- Method 2
  ```python
  from zpy.awm import * #awm:Always With Me
  nowf('d')
  ```
